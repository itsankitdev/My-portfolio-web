## ✨ About

A modern, interactive 3D portfolio website built with **React + Vite**, featuring smooth animations, glassmorphism design, and a deep purple neon theme.

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| ⚛️ React + Vite | Frontend Framework |
| 🎨 Tailwind CSS | Styling |
| 🎭 Framer Motion | Animations |
| 🌐 Three.js | 3D Background |
| 📦 Vercel | Deployment |

---

## 🎯 Features

- ✅ 3D Animated Background (Three.js blob)
- ✅ Smooth page load & scroll animations (Framer Motion)
- ✅ Glassmorphism UI design
- ✅ Deep Purple + Pink Neon theme
- ✅ Fully Responsive (Mobile + Desktop)
- ✅ Hamburger menu with hover effects
- ✅ Project cards with GitHub & Live preview buttons
- ✅ Animated Navbar with underline hover effect
- ✅ Contact section with direct email & social links

## 🚀 Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/itsankitdev/My-portfolio-web.git

# 2. Go into the folder
cd My-portfolio-web

# 3. Install dependencies
npm install

# 4. Start dev server
npm run dev
```

Open **http://localhost:5173** in your browser.
